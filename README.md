  **Data Field Survey Analysis – Power BI Project**

**PROJECT OVERVIEW**

This project is focusing on getting insights from data taken from survey on X (formerly Twitter) app by (Alex the analyst) . Many data fields workers and students participated in answering various questions related to data fields like ( salary ,work/life balance , average age , etc..) .  

 **PROJECT OBJECTIVES**
•	Analyze survey responses from the data field.
•	Identify trends in tools, salaries, roles and gender . 
•	Understand preferences and career paths in the data industry.

**DASHBOARD FEATURES**
•	Salary analysis by role
•	Favorite programming languages
•	Job satisfaction metrics
•	Work-life balance insights
•	Geographic distribution of respondents

**DATASET INFORMATION**
Source: Data Professionals Survey

**EXAMPLE FIELDS**
•	Job Title
•	Country
•	Salary
•	Programming Language
•	Work Satisfaction
•	Experience Level

**TOOLS & TECHNOLOGIES**
•	Power BI
•	Excel / CSV Dataset

**DATA CLEANING PROCESS**
Steps performed before visualization:
•	Handling missing values
•	Standardizing job titles
•	Removing duplicates
•	Transforming salary fields
•	Changing data types

**KEY INSIGHTS**
•	Data Scientists reported the highest average salaries.
•	Python is the most preferred programming language.
•	Work-life balance varies significantly across roles.
•	average salary is almost the same for both genders
•	Happiness with salary is low .
